📋 Fitness Planner Local Website
--------------------------------
این نسخه برای اجرای محلی طراحی شده است.

📦 شامل:
- index.html
- پوشه‌های scripts و styles
- سایر فایل‌های مرتبط با وب‌اپ

🚀 روش اجرا:
1. فایل زیپ را از حالت فشرده خارج کنید.
2. وارد پوشه‌ی استخراج‌شده شوید.
3. یکی از روش‌های زیر را انتخاب کنید:

🔹 روش ۱ (با Python 3):
   python -m http.server 8080
   سپس در مرورگر باز کنید:
   http://localhost:8080

🔹 روش ۲ (با Node.js نصب‌شده):
   npx serve .
   یا:
   npm install -g serve
   serve

📌 نکته:
- برای عملکرد کامل JavaScript مطمئن شوید که از اجرای مستقیم (file://) استفاده نکنید.
- اگر فایل‌ها به منابع بیرونی (API یا اینترنت) وصل هستند، باید آنلاین باشید.
